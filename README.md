🍽️ Recipe Sharing Website (PHP, MySQL)

A dynamic Recipe Sharing Web Application built using PHP, HTML, CSS, JavaScript, and MySQL. Users can explore and purchase recipe PDFs, while admins manage recipes, users, and payments through a powerful dashboard.

🚀 Features
👤 User Panel
🔍 Browse Recipes – Explore available recipes with images and details
🔎 Search Recipes – Find recipes by name or category
❤️ Favorites – Save recipes for later
🛒 Add to Cart – Add recipes to a shopping cart
💳 Buy Recipe PDFs – Purchase downloadable recipe files

🛠️ Admin Panel
➕ Add Recipe – Upload new recipes with PDF files
✏️ Manage Recipes – Edit or delete recipes
👥 Manage Users – View, block, or delete users
💬 Feedback Management – Read user feedback
💰 Payment Confirmation – Verify Khalti transactions
📧 Email Notifications – Send emails to users

🧰 Tech Stack
Frontend: HTML, CSS, JavaScript
Backend: PHP (Core PHP)
Database: MySQL
Payment Gateway: Khalti
